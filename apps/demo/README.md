# @rtl-safe-ai/demo

A small Vite + React playground for experimenting with the `@rtl-safe-ai/core`
engine.

> Status: **scaffold** — UI is wired to the placeholder core API.

## Run

```bash
npm run dev:demo
# or
npm run dev --workspace @rtl-safe-ai/demo
```

The demo is fully local: a strict Content-Security-Policy blocks third-party
scripts and the app makes no network calls.
