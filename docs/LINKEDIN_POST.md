## English (concise)

I built **RTL Safe AI** — an open-source, security-first engine that fixes
right-to-left text (Hebrew, Arabic, and more) in AI chat interfaces.

If you write Hebrew or Arabic with ChatGPT, Claude, or Gemini, you've seen it:
mix in an English word, a link, or a code snippet and the line flips into an
unreadable mess. RTL Safe AI detects the correct direction, keeps code and math
left-to-right, and neutralizes invisible/"Trojan Source" Unicode tricks.

What it deliberately does NOT do: no telemetry, no analytics, no network
requests. Everything runs locally in your browser.

The repo includes:
• A pure TypeScript core engine (zero runtime dependencies)
• A Manifest V3 browser extension (minimal permissions)
• A React demo
• 111 tests, CI, CodeQL, and a full security/privacy documentation layer

It's a personal open-source project, not a certified or audited product — but
it's built to align with recognized secure-development principles.

Feedback and contributions welcome 🙏
